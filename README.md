# Eternite game

## Building

### Prerequisites: (for now Windows, later Linux too)
- CMake
- Visual Studio 2019 

### Steps
- git clone --recursive (Submodules!!!)
- mkdir Build
- cd Build
- cmake ..
- cmake --build . --target Client
- Navigate to: Build\GameClient\Debug\Client.exe

Or use your fancy IDE (VSCode Cmake Extension)
